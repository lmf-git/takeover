00 node_modules/

Zero dependencies

"dependencies": {} — and we mean it. No Vite, no Webpack, no React, no ws, no chokidar. The framework runs on Node built-ins.

01

SSR + Declarative Shadow DOM

Server-rendered output uses <template shadowrootmode="open">. The browser attaches shadow roots before JS runs. Hydration is event binding — never DOM replacement.

02 fs.watch

Native ESM dev server

The browser fetches modules directly — no transpile step. WebSocket-based HMR with a 50ms debounce. Three strategies: CSS hot-swap, full reload, component re-import.

03

File-system routing

app/About/ → /about. app/Users/[id]/User/ → /users/:id/user. Click handlers on <a route> intercept and push state. Lifecycle hooks: beforeEach, afterEach, onError.

04 EventTarget

Reactive store

Proxy over EventTarget. store.on('counter', fn) returns unsubscribe. Components declare static store = ['user', 'theme'] and re-render automatically.

05

Built-in i18n

EN · ES · FR included. Accept-Language detection on the server. Messages are baked into __INITIAL_STATE__ — no flash, no flicker, no client-side fetch.

06

Bundler & minifier

Pure-Node ESM bundler traces static imports into a single IIFE. Character-level tokenizer handles template literals, regex, and comments. ~35–50% reduction.

07

Edge-ready

Cloudflare Pages Worker and Netlify Functions targets. Content-addressed assets (core.d6d6fad4.js) for long-lived caching. SSR at the edge, hydration in the browser.
